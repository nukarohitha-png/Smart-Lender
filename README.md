# Smart Lender – Loan Approval Prediction System

## 📌 Project Overview
Smart Lender is a machine learning-based web application that predicts whether a loan application is likely to be approved based on applicant details.

The application is built using Flask and a trained machine learning model.

---

## 🚀 Features

- User-friendly web interface
- Loan approval prediction
- Machine Learning model integration
- Fast prediction results
- Responsive design

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Scikit-learn
- Pandas
- NumPy

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/nukarohitha-png/Smart-Lender.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
